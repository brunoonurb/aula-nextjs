npx create-next-app@latest  aula-node --ts --app --tailwind --eslint --src-dir --import-alias "@/*" --use-npm --no-turbopack --yes


| `aula-node` | paste/nome projeto |
| `--ts` | Usa TypeScript |
| `--app` | App Router (`src/app/`) |
| `--tailwind` | Configura Tailwind CSS |
| `--eslint` | Adiciona ESLint |
| `--src-dir` | Coloca o código em `src/` |
| `--import-alias "@/*"` | Atalho de import `@/...` |
| `--use-npm` | Usa npm como gerenciador |
| `--no-turbopack` | Usa o bundler padrão |
| `--yes` | Aceita os defaults sem perguntar |
